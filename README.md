# Markus

Dein persönlicher AI-Assistent – gehostet auf GitHub Pages.

## Tech Stack

- **Next.js 15** (App Router)
- **Tailwind CSS v4**
- **TypeScript**
- **GitHub Pages** (via GitHub Actions)

## Entwicklung

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Automatisch via GitHub Actions bei jedem Push auf `main`.
